# Master Checklist

Use this file as the top-level control list for the whole report.

## Global checks
- [ ] Title is consistent with the final system scope.
- [ ] The report story is consistent from introduction to conclusion.
- [ ] Every main claim has evidence or is clearly marked as preliminary.
- [ ] The writing style is simple, academic, and easy to follow.
- [ ] Terminology is stable across the whole report.
- [ ] All unfinished parts are marked clearly.
- [ ] Figures and tables are referenced in text.
- [ ] Citations are complete and formatted consistently.
- [ ] The final report does not claim results that are not ready yet.

## Scope checks
- [ ] Retrieval pipeline is described clearly.
- [ ] GraphRAG is included as part of the project scope.
- [ ] UI and full QA application are included as part of the project scope.
- [ ] LLM response generation is included as part of the project scope.
- [ ] Custom loss design is included if it is part of the final implementation.
- [ ] Pending work is separated from completed work.
