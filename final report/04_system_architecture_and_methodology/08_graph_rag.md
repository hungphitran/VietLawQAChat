# GraphRAG

GraphRAG is part of the project scope and should be described as a retrieval extension that uses graph structure to expand and organize evidence. In the legal domain, graph edges can represent citation links, related articles, amendments, or semantic relations between legal passages. This gives the system a way to move beyond flat passage ranking.

At the current stage, the graph-based component should be presented carefully if the implementation is still under development. The report can describe the intended graph schema, the planned traversal logic, and the role of graph expansion in the final QA pipeline. If some parts are already implemented only as a prototype, they should be labeled clearly as prototype-level work.
