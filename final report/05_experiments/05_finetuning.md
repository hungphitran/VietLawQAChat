# Fine-tuning Experiments

The codebase already includes training support for both the bi-encoder and the cross-encoder, so fine-tuning is part of the planned experimental story. The bi-encoder training loop uses a ranking objective, while the cross-encoder training loop builds positive and negative pairs for more precise scoring.

If some fine-tuning runs are still ongoing at the time of writing, the report should separate them from the completed baseline experiments. In that case, it is better to report the training setup and the intended evaluation plan than to invent a final result that is not ready yet.
