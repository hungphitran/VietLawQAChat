# GraphRAG Experiments

GraphRAG experiments should be reported only if the graph module is sufficiently stable to support a fair evaluation. If the graph layer is already usable, the experiments should compare retrieval with and without graph expansion, and should also check whether graph traversal helps answer grounding.

If the GraphRAG module is still a prototype, the report should keep this subsection short and honest. It is still useful to describe the intended experiment design, but the section should not present incomplete results as final evidence.
