# End-to-End Experiments

The end-to-end experiment should test the full path from user query to final answer. This is the most complete evaluation because it checks whether the retrieval modules, graph expansion, and generation stage work together as one system. When available, the report should compare answer quality with and without retrieval improvement stages.

If the full QA pipeline is not yet finished, this section can still describe the intended evaluation protocol and the current implementation status. The key point is that the final project is more than a retriever: it is a legal QA system built on top of retrieval.
