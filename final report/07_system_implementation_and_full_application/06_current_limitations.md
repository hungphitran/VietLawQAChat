# Current Limitations

The current system still has several limits. Some modules are already implemented but still need final tuning, while others are present as planned extensions or prototypes. In particular, GraphRAG integration, custom loss exploration, and the final LLM response layer may still need more work before the application can be treated as fully finished.

This limitation is not a weakness of the report; it is part of the actual project status. A clear report should separate completed modules from in-progress modules so that the reader can understand both the current achievements and the remaining engineering work.
