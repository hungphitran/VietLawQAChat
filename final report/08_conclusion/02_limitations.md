# Limitations

The current system is still incomplete in a few places. Some advanced components, especially GraphRAG integration and the final answer generation layer, may still be under active development. In addition, the best fine-tuned results may still need final training runs and ablation checks before they can be treated as final numbers.

These limits should be read as scope boundaries rather than failures of the core idea. The retrieval architecture itself is already useful, but the full legal QA system still needs more integration work before it can be considered fully polished.
