# Final Report Workspace

This folder is the drafting workspace for the final technical report.

Writing rules:
- Keep the style paper-like, clear, and direct.
- Use common academic English with simple vocabulary.
- Keep one paragraph focused on one idea.
- Do not overstate results that are not ready yet.
- Mark unfinished parts as `TODO`, `Prototype`, or `Planned in scope`.

Suggested report order:
1. Abstract
2. Introduction
3. Related Work
4. Dataset and Data Processing
5. System Architecture and Methodology
6. Experiments
7. Results and Discussion
8. System Implementation and Full Application
9. Conclusion
